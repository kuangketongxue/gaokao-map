# 🎓 2026高考知识点 3D 思维导图

这是一个基于 GitHub Pages 托管的、具有 Obsidian 风格的 **3D 交互式关系图谱**。旨在通过立体的视觉化方式，梳理高考核心知识点及其跨学科联系。

### 🌟 项目特色
- **3D 视角**：支持旋转、缩放、平移，全方位查看知识网络。
- **Markdown 支持**：点击知识点节点，即可在右侧查看精美的 Markdown 格式解析。
- **导航分类**：顶部导航栏支持按 **语文、数学、英语、政治、物理、化学** 快速筛选节点。
- **APL 测试**：内置测试入口跳转功能。
- **自定义背景**：支持自由更换背景图，营造沉浸式学习氛围。

---

### 📂 文件结构说明
```text
├── index.html            # 项目入口文件（包含 3D 引擎及逻辑）
├── 1.html                # APL 测试页面
├── assets/
│   └── bg.jpg            # 背景图片（在此处替换你喜欢的图片）
└── data/
    ├── nodes.json        # 知识点定义文件（包含 ID、科目、名称、Markdown内容）
    └── links.json        # 知识点关系文件（定义点与点之间的连线）
```

---

### 📝 如何添加/编辑知识点

#### 1. 编辑知识点 (`data/nodes.json`)
在该文件中添加新对象。注意 `content` 字段支持 **Markdown** 语法，但换行请使用 `\n`。
```json
{
  "id": "m_001",
  "group": "数学",
  "name": "复合函数求导",
  "content": "# 复合函数求导\n\n**法则：** $f[g(x)]' = f'(u) \cdot g'(x)$\n\n- 步骤1：分解内外层\n- 步骤2：由外向内层层剥皮"
}
```

#### 2. 编辑关系线 (`data/links.json`)
在该文件中定义哪些知识点需要连起来。`source` 和 `target` 指向节点的 `id`。
```json
{
  "source": "m_001", 
  "target": "physics_001" 
}
```

### 🖼️ 如何修改背景图片
1. 准备一张你的高清大图（推荐 1920x1080）。
2. 将图片命名为 `bg.jpg`。
3. 上传并覆盖 `assets/` 文件夹下的旧文件。
4. 刷新网页即可生效（若未生效，请尝试清除浏览器缓存）。

---

### 🚀 快速部署到 GitHub Pages
1. 创建一个新的 GitHub Repository。
2. 将本项目所有代码上传至仓库。
3. 进入 **Settings** -> **Pages**。
4. 在 **Build and deployment** 中，Source 选择 `Deploy from a branch`。
5. Branch 选择 `main` (或 `master`)，保存。
6. 几分钟后，即可通过 `https://你的用户名.github.io/仓库名/` 访问。
