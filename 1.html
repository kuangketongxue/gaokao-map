⚠️ 遇到连接失败 / 无响应？点击查看解决方案

问题原因：浏览器的 CORS 安全策略阻止了跨域请求。本地 HTML 文件请求外部 API 时，如果服务端没有返回 Access-Control-Allow-Origin 头，请求会被拦截。

解决方案（任选其一）：

1️⃣ 上方选择一个 CORS 代理（推荐先试 corsproxy.io）

2️⃣ 安装浏览器插件：CORS Unblock 或 Allow CORS

3️⃣ 用命令行启动无安全限制的 Chrome：
chrome --disable-web-security --user-data-dir=/tmp/chrome_dev

4️⃣ 确认你的中转站 URL 是否正确（OpenAI 格式一般需要以 /v1 结尾）
